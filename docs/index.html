JJWarehouse is a full-stack inventory management system developed for Android using Java and Android Studio. What began as a basic local CRUD app has grown into a complete, secure, and scalable inventory solution with user authentication, cloud-based storage, real-time search functionality, and a thoughtful, touch-optimized UI.

This project was built entirely from the ground up ‚Äî every screen, every database table, every line of backend logic. It reflects my process of relearning mobile development by doing, with a focus on writing secure, efficient code and building apps that solve real problems.

üß≠ Project Background

This application started as a simple academic exercise: a local inventory tracker with basic Create, Read, Update, and Delete (CRUD) functionality powered by a SQLite database. Users could manage item names, IDs, and quantities on their device, but there was no user account system, no remote data access, and limited security.

I decided to treat this as an opportunity to challenge myself and turn the prototype into something production-worthy. I gradually rebuilt and expanded the app, touching nearly every layer of the stack.

üîê Security & Authentication

  • Passwords are hashed using bcrypt before storage
  • JWT (JSON Web Token) used for secure session management
  • Tokens stored with SharedPreferences locally
  • Real-time login feedback and server-side validation

üß∞ Backend Development

The RESTful API was built with:

  • Node.js + Express.js
  • MySQL (AWS RDS)
  • CORS + JWT middleware
  • Input validation + custom error handling

Server communication is handled via Retrofit. Support classes include ApiInterface, RetrofitClient, ItemResponse, and LoginResponse.
